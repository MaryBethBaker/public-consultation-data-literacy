Learning Modules Overview

The content of this guide is divided into a series of learning modules. Each learning module covers a step in a generalized data processing workflow.

Please note: As this guide is currently in the prototype phase, only the first two modules have been developed.

Module 1: What is Public Consultation Data?

This module introduces the concept of public consultation results as data. You will see examples of different kinds of public consultation data and be introduced to some basic data types. You will also get to take a look at some example datasets and explore the types of data that they contain.

Go to this module

Module 2: Data Preprocessing

This module will teach you about basic data preprocessing. Preprocessing is important as it allows you to improve the quality of your data, making it more suitable for analysis and thus able to provide you with greater insight. You will learn about different categories of preprocessing activities and get to explore a real public consultation dataset. You will also be asked to perform different preprocessing tasks on a variety of open consultations datasets from open.canada.ca.
